# Career OS

Career Operating System for professional development, project management, and continuous learning.
## Overview

Career OS is my structured career management system designed to organize professional experience, learning, projects, and long-term career development.
Rather than keeping information across multiple documents, this project follows a Single Source of Truth (SSOT) approach where career planning, learning, and portfolio development work together.
## Objectives

- Organize career development
- Track technical learning
- Build portfolio projects
- Document continuous improvement
- Support long-term career growth
